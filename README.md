## ğŸ”§ Customization

Want to customize DICE? It's easy!

### Change Colors
Edit these variables in `css/style.css`:
```css
:root {
    --primary-bg: #000000;      /* Background */
    --primary-text: #ffffff;    /* Text */
    --accent: #6366F1;          /* Main accent color */
    --success: #10b981;         /* Yes color */
    --danger: #ef4444;          /* No color */
    --warning: #f59e0b;         /* Maybe color */
}
```

### Add New Tools
1. Add a new section in `index.html`
2. Add corresponding CSS in `css/style.css`
3. Add JavaScript logic in `js/script.js`
4. Add a tab button in the navigation

### Modify Animations
All animations are defined in `css/style.css`. Search for `@keyframes` to customize.
