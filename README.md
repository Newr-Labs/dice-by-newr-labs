
# DICE by NEWR Labs
![Logo](https://picsum.photos/200/200?grayscale&blur=2)

**Let fate decide.**

DICE is a minimal, cyber-modern suite of random decision-making tools built for clarity and speed. Whether you need to settle a debate, pick a winner, or generate numbers, DICE provides a premium, distraction-free experience.

## Live Demo
Check it out live: [https://newr-labs.github.io/dice-by-newr-labs](https://newr-labs.github.io/dice-by-newr-labs)

## Download & Run
### For normal users:
1. Click **Code → Download ZIP**.
2. Extract the ZIP file.
3. Open `index.html` in your web browser.
4. Works entirely offline.

### For developers:
```bash
git clone https://github.com/Newr-Labs/dice-by-newr-labs.git
cd dice-by-newr-labs
# Open index.html in your browser or serve with a static server
```

## Features
- **Dice Roll**: Roll 1–6 dice with animated effects and sum calculation.
- **Coin Flip**: Heads or tails with a high-fidelity flip animation.
- **Yes / No / Maybe**: Instant answers for your binary (or trinary) questions.
- **Random Number Generator**: Customizable ranges with a one-tap copy feature.
- **Spin the Wheel**: Text-based wheel that highlights choices as it cycles.
- **Random Picker**: Select items from a custom list with an optional "remove after pick" toggle.

## Tech Stack
- **React 18**: Modern UI logic.
- **TypeScript**: Robust type safety.
- **Tailwind CSS**: Utility-first styling for high performance and sleek looks.
- **No Build Step**: Purely static deployment ready for GitHub Pages.
