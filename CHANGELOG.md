
# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-24
### Added
- Initial release of DICE by NEWR Labs.
- Six core tools: Dice, Coin, Yes/No, RNG, Wheel, and Picker.
- AMOLED black theme with Indigo accents.
- Responsive mobile-first design.
- Smooth CSS animations for all interactions.
- One-tap copy functionality for RNG.
- List management for Picker and Wheel.
